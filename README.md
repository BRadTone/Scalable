# Scalable

Boilerplate for simple web pages.
Main goal - achive usefull and scalable sass files architecture.
## Getting Started

SCSS structure
<pre>
├── __GLOBAL            // @include(<all files>)
├── _base

├── _components 
|	├── _buttons.scss
|	├── _inputs.scss
|	├── _navigations.scss
|	└── _popUp.scss
├── _components 
|	├── _buttons.scss
|	├── _inputs.scss
|	├── _navigations.scss
|	└── _popUp.scss
├── _helpers 
|	├── _functions.scss
|	├── _mixins.scss
|	├── _placeholders.scss
|	└── _variables.scss
├── _vendor
	└──[third-party-libs]
├── _pages
	└── _index.scss
</pre>
